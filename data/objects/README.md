# Objects dataset (flat, non-hierarchical)

CSV format: `category,label,aliases`

- **category**: e.g. Sports, Animals, Fruits
- **label**: main display name
- **aliases**: optional, semicolon-separated alternatives (e.g. `Plane` for Airplane)

## Files

- `objects_test.csv`: 34 objects, quick experiments
- `objects_full.csv`: 158 objects, complete benchmark
